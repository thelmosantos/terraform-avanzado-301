## Qué aprenderás

- Generar infraestructura dinámica con `count` y `for_each`.
- Usar `locals`, expresiones condicionales y funciones integradas.
- Manipular mapas y listas para parametrizar recursos.

## Contexto

- En lugar de copiar bloques, dejas que el código genere los recursos.
- `for_each` por defecto (claves estables); `count` solo para on/off o listas fijas.
- Trabajo **local**: todo se valida con `plan`, sin aplicar.
