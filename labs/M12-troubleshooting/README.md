## Qué aprenderás

- Interpretar mensajes de error de Terraform y de los providers.
- Usar `TF_LOG` para depurar el comportamiento de un provider.
- Diagnosticar y recuperar escenarios típicos: dependencias circulares, state corrupto, drift, variables incorrectas.

## Contexto

- Cuando algo peta, es el ejercicio más útil del curso: el error suele decir la verdad.
- Método: lee el error completo, reproduce, aísla, corrige.
